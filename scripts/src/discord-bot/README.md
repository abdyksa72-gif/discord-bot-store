# 🤖 بوت Discord الاحترافي

بوت Discord متكامل باللغة العربية مع ميزات الإشراف، الترحيب، التذاكر، وأكثر.

## ⚡ خطوات التثبيت

### 1. إنشاء البوت في Discord
1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. اضغط **New Application** وسمّها
3. اذهب إلى **Bot** ← **Add Bot**
4. انسخ **Token** (احتفظ به سري!)
5. فعّل هذه الصلاحيات في **Privileged Gateway Intents**:
   - ✅ Server Members Intent
   - ✅ Message Content Intent

### 2. دعوة البوت لسيرفرك
اذهب إلى **OAuth2** ← **URL Generator** واختر:
- Scopes: `bot`
- Permissions: `Administrator` أو اختر الصلاحيات يدوياً

### 3. تثبيت المتطلبات
```bash
npm install discord.js
```

### 4. إعداد التوكن
```bash
# على Windows
set DISCORD_TOKEN=توكن_البوت_هنا

# على Linux/Mac  
export DISCORD_TOKEN=توكن_البوت_هنا
```

### 5. تشغيل البوت
```bash
node index.js
```

## 📁 إعداد قنوات السيرفر

أنشئ هذه القنوات للاستفادة من جميع الميزات:
| القناة | الغرض |
|--------|--------|
| `الترحيب` | رسائل ترحيب الأعضاء الجدد |
| `سجل-الأحداث` | تسجيل إجراءات الإشراف |
| `التذاكر` (فئة) | فئة قنوات التذاكر |

## 📋 الأوامر المتاحة

### 🛡️ الإشراف (للمشرفين فقط)
| الأمر | الوصف | مثال |
|-------|--------|-------|
| `!kick @عضو [سبب]` | طرد عضو | `!kick @user سلوك مسيء` |
| `!ban @عضو [سبب]` | حظر عضو | `!ban @user إزعاج متكرر` |
| `!mute @عضو [دقائق]` | كتم عضو | `!mute @user 30` |
| `!clear [عدد]` | مسح رسائل (1-100) | `!clear 50` |

### 📊 المعلومات
| الأمر | الوصف |
|-------|--------|
| `!help` | عرض قائمة الأوامر |
| `!ping` | فحص سرعة البوت |
| `!userinfo [@عضو]` | معلومات مستخدم |
| `!serverinfo` | معلومات السيرفر |

### 🎫 نظام التذاكر
| الأمر | الوصف |
|-------|--------|
| `!ticket [موضوع]` | فتح تذكرة دعم |

### 🎉 الترفيه
| الأمر | الوصف | مثال |
|-------|--------|-------|
| `!8ball [سؤال]` | كرة 8 السحرية | `!8ball هل سأنجح؟` |
| `!roll [رقم]` | رمي النرد | `!roll 20` |

## ✨ الميزات التلقائية

- **الترحيب**: ترحيب تلقائي بالأعضاء الجدد مع معلوماتهم
- **سجل المغادرة**: تسجيل مغادرة الأعضاء تلقائياً
- **سجل الإجراءات**: تسجيل جميع إجراءات الإشراف
- **نظام التذاكر**: قنوات خاصة لكل تذكرة مع زر الإغلاق
